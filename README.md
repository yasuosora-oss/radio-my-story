# 📻 Radio My Story

家族や友達とスマホを回しながら、即興の物語をリレー録音すると、
AIがそれをひとつの**ラジオドラマ**（多声優の音声＋BGM＋効果音つき）に仕上げてくれるWebアプリです。

## あそびかた

1. 参加人数（2〜5人）と、Gemini APIキーを入力してスタート
2. スマホを回しながら、ひとりずつ物語のつづきを録音（何周でもOK）
3. 「物語をここで完結する」→ AIがあらすじにまとめる
4. ジャンル（恋愛・SF・ミステリー・ホラー・コメディ・時代劇）を選ぶ
5. タイトルコールつきの音声ドラマが、BGM・効果音と一緒に再生されます
6. 同じ録音のまま「別のジャンルでおかわり」もできます

## 利用前に知っておいてほしいこと

- **録音した声は、物語の生成のためにGoogleのAI（Gemini API）に送信されます。**
  参加する人にはあらかじめ伝えてください。
- あらすじ・台本・タイトル・音声は**AIが生成したもの**です。内容は再生前後にご確認ください。
- 利用には各自の **Gemini APIキー** が必要です（Google AI Studioで無料発行できます）。
  キーはこのアプリのコードには含まれず、ブラウザからGoogleへの通信にだけ使われます。
- AIの利用回数には上限があります（無料キーは特に音声生成が少なめです）。

## 動作環境

スマートフォン（Chrome / Safari）またはPCのブラウザ。マイクの使用許可が必要です。
